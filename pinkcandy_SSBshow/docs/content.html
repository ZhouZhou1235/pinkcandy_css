闪星方块 网页内容

网页内容

屏幕

screenFull 标准屏幕
screen 屏幕

文本

infoBox 信息盒子
-- 显示基础的网页信息，搭配h1 h2 p使用。
txtBox 文本盒子
-- 用于显示文本 搭配h1 h2 p pre使用。
boardBox 文本盒子容器
-- 显示多组文本的区域，搭配txtBox使用。

媒体

galleryBox 画廊盒子容器
-- 展示多个媒体的区域，需要展示的媒体内容统一放到这里。
showGalleryIndex 画廊盒子
-- 在画廊盒子容器中的媒体，搭配h1 h2 p img 使用。
showGallery 画廊大盒子
-- 在网页中展示单个媒体。
---- firstSide 媒体放置区，如图片、视频等
---- secondSide 信息说明区
-------- txtBox
------------ information 在此处可以显示有字段编排的文本
previewGallery 预览画廊盒子
-- 媒体预览的小框
---- galleryTitle 在小框中心显示一段文本
---- imgBox 媒体需放置在此处
showPic 图片展示盒子

交互

formBox 表单盒子
-- 用户输入的区域。
---- side 可以分成左右两块
userBox 用户盒子
-- 网站用户的显示区域
---- leftSide 左空间，搭配galleryBox使用。
---- right 右空间
-------- essayBox 爪记盒子
---------- 此处可以展示用户的动态信息，搭配h1 h2 p使用。
userBoxIndex 首页用户盒子
-- 在首页显示登录用户信息
operateBox 操作盒子
-- 用户使用网站功能的区域
---- information 描述功能
---- contentBox 操作内容盒子
------ 聚焦到此处时contentBox将自动展开。
statementBox 信息报告盒子
-- 显示一段通知，提示或警告。
commentsBox 评论盒子
-- 显示用户的评论信息，搭配h1 h2 p使用。
