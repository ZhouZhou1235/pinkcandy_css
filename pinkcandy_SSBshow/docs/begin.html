闪星方块 我的第一个网页

首先获取闪星方块

在头标签中添加闪星方块
SSB_default.css 默认
SSB_white.css 白
SSB_black.css 黑

布置包含所有元素的屏幕screen

class选择器
screenFull 标准屏幕
screen 有导航栏的屏幕

创建导航栏表示网页主题

id选择器
menuFull 标准导航栏框
menu 侧边导航栏框

添加基本的文字信息

class选择器
infoBox 信息盒子

CSS

<head> <link rel="stylesheet" href="https://pinkcandy.top/css/SSB_default.css"> </head>

HTML

<body> <div class="screenFull"> <div id="menuFull"> <span>我的第一个闪星方块网页</span> </div> <div class="infoBox"> <h1>粉糖粒子</h1> <h2>幻想动物画廊</h2> <p>详细信息......</p> </div> </div> </body>
