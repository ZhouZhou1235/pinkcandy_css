闪星方块是粉糖粒子网站使用的样式框架，开发者为pinkcandyzhou。 闪星方块旨在以最简单的方式完成网页的布局和装饰。 通过使用闪星方块可以轻松地装饰网页。

获取框架

链接到粉糖粒子服务器 <link rel="stylesheet" href="https://pinkcandy.top/css/SSB_default.css"> <link rel="stylesheet" href="https://pinkcandy.top/css/SSB_white.css"> <link rel="stylesheet" href="https://pinkcandy.top/css/SSB_black.css"> 或下载源码 GitHub
